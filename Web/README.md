- 동일 출처 정책이나 CORS에서의 출처란 무엇인지 말씀해주세요.

- 브라우저의 기본 동작 과정에 대해 말씀해주시고 CORS 작동 방식 3가지 시나리오 중 1가지에 대해 자세히 말씀해주세요.
