- 해시 충돌이 발생하는 이유와 충돌 해결 전략(예: 체이닝, 개방 주소법)에 대해서 간략히 설명해주세요.

- 힙과 시간 복잡도에 대해 설명해주세요.

- 그래프 구현법 2가지에 대해 설멍해주세요. 그리고 두 방법의 장단점에 대해 비교해주세요.

- 트리 탐색법 2가지 중 1개에 대해 설명해주시고 어느 상황에서 사용하는 것이 적절한지 설명해주세요.
